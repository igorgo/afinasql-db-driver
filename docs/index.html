▸ execute(aSessionId, aSql, aBindParams = [], aExecuteOptions = {}, aConnection = null)

Executes a statement

execute(aSessionId: string, aSql: string, aBindParams: (OraSqlParams | Array)?, aExecuteOptions: ({} | oracledb.IExecuteOptions)?, aConnection: oracledb.Connection?): Promise<oracledb.IExecuteReturn>
Parameters
aSessionId (string) An Afina Sequel Session ID
aSql (string) The SQL string that is executed. The SQL string may contain bind parameters.
aBindParams ((OraSqlParams | Array)? = []) Definintion and values of the bind parameters. It's needed if there are bind parameters in the SQL statement
aExecuteOptions (({} | oracledb.IExecuteOptions)? = {}) Execution options o control statement execution, such a fetchInfo, outFormat etc.
aConnection (oracledb.Connection? = null) Existing connection. If it is set, then connection won't be closed, if not set the new connection will be open and will be closed after execution
Returns
Promise<oracledb.IExecuteReturn>: The result Object. See https://github.com/oracle/node-oracledb/blob/master/doc/api.md#-result-object-properties
